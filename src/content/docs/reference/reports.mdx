Reports require the `reports` scope, which is separate from `read` and `write`.

## AR Aging

```
GET /api/v1/books/reports/ar_aging
```

**Scope:** `reports`

Returns accounts receivable aging buckets with invoice-level detail.

### Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `as_of` | date | today | Aging date |
| `currency` | string | USD | Currency filter |

### Response

```json
{
  "data": {
    "as_of": "2026-03-12",
    "currency": "USD",
    "buckets": {
      "current": [
        {
          "contact_id": 42,
          "contact_name": "Acme Corp",
          "invoice_id": 123,
          "invoice_number": "INV-00001",
          "issue_date": "2026-03-01",
          "due_date": "2026-04-01",
          "total_cents": 50000,
          "balance_due_cents": 50000
        }
      ],
      "1_30": [],
      "31_60": [],
      "61_90": [],
      "91_plus": []
    },
    "totals": {
      "current": 50000,
      "1_30": 0,
      "31_60": 0,
      "61_90": 0,
      "91_plus": 0

```

### Aging Buckets

| Bucket | Description |
|---|---|
| `current` | Not yet past due |
| `1_30` | 1-30 days past due |
| `31_60` | 31-60 days past due |
| `61_90` | 61-90 days past due |
| `91_plus` | 91+ days past due |

### Caching

This endpoint is cached for 60 seconds. The cache automatically invalidates when invoices or payments change.
